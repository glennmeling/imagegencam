# ImageGenCam 3D Model

This folder contains the 3D-printable ImageGenCam model files. You can print them yourself, or send them off to a print shop. We recommend PETG, as it works best with the folding structure of the camera, but PLA should be okay too. This has not been tested with resin printing. 

- `stl/` contains ready-to-print STL exports.
- `step/` contains editable STEP source files.

Parts:

- `Folding Case`
- `Clip On Lens`
- `Buttons - Top`
- `Buttons - Arrows`
- `Buttons - Circle Square`
