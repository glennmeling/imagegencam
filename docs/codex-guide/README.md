# Codex Guide References

These files are for Codex when it is guiding someone through the ImageGenCam build from this repo. Users should start with the main `README.md`; Codex should load only the reference needed for the current step.

- `kit-flow.md`: prebuilt kit setup
- `setup-flow.md`: DIY setup from a blank microSD card
- `remote-codex.md`: SSH and Codex Desktop remote connection
- `sudo-and-terminal.md`: password, sudo, and Terminal handoff
- `hardware-checks.md`: camera, display, PiSugar, and network checks
- `troubleshooting.md`: focused diagnostics when something fails
